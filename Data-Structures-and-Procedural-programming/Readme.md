📘 Overview

This project contains two algorithmic problems implemented using arrays.
Each problem demonstrates basic programming logic, array manipulation, and parameter passing.

🧩 Problem 1 – Sum of Distinct Elements from Two Sets
📝 Description
Given two sets (or arrays) of integers, the goal is to find the sum of all distinct elements — meaning the elements that appear in only one of the two sets.
Set 1: [3, 1, 7, 9]
Set 2: [2, 4, 1, 9, 3]
The distinct elements are 4, 7, and 2.
Sum = 4 + 7 + 2 = 13
🧠 Algorithm Steps

## Initialize a variable sum = 0.

## For each element in Set 1, check if it is not present in Set 2.

## If not found, add it to sum.

## Repeat the same process for Set 2, checking against Set 1.

## Finally, print or return the total sum.

🔢 Problem 2 – Dot Product & Orthogonality Check
📝 Description
This problem defines a procedure and a function to compute the dot (scalar) product of two vectors.
It then checks whether the two vectors are orthogonal.
Two vectors are orthogonal if their dot product equals zero.

🧠 Part 1: Procedure dot_product
Input: Two arrays (vectors) v1 and v2
Output: A scalar value ps representing the dot product

⚙️ Part 2: Function Version
If dot_product is written as a function, the algorithm changes slightly
